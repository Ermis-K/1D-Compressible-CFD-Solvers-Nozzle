# 1D-Compressible-CFD-Solvers-Nozzle
A computational fluid dynamics (CFD) implementation for simulating compressible flow through converging-diverging nozzles using the MacCormack method.

* **`First_Non_Conservative.m` & `Second_Non_Conservative.m`**: Isentropic Subsonic-Supersonic Expansion (Non-Conservative Form)
* **`First_Conservative.m` & `Second_Conservative.m`**: Isentropic Subsonic-Supersonic Expansion (Conservative Form)
* **`Third_Non_Conservative.m`**: Standing Normal Shock (Non-Conservative Form with Shock Capturing)
* **`Third_Conservative.m`**: Standing Normal Shock (Conservative Form with Shock Capturing)
* **`Third_Shock_Fitting_Fail.m`**: Standing Normal Shock (Shock Fitting Method - Failed Implementation)
